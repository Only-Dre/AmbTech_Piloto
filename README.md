## AmbTech
### Situação de Aprendizado

Utilizando de API ThingSpeak, MySQL, Docker e Flask Framework, foi realizado um projeto de leitura de Temperatura e Umidade (simulados) com o intuito de:
• capturar e transmitir dados;
• armazenar informações localmente;
• exibir dados de forma organizada para tomada de decisão;
• utilizar contêineres para facilitar implantação e manutenção.

Dessa forma, apresentamos graficamente de duas formas diferentes - tanto pelo próprio ThingSpeak - quanto pelo Local.
